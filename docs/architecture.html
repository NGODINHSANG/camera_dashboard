🔄 Auto-Record Flow

  1. Camera push stream → MediaMTX nhận RTMP tại :1935
  2. MediaMTX webhook → Gọi /api/internal/webhooks/stream-ready
  3. Backend check → Camera có auto_record=1?
  4. Start FFmpeg → Record HLS stream → MP4 (15 phút/segment)
  5. Stream ngắt → Webhook stream-not-ready → Stop recording
  6. HLS Converter → Convert MP4 → HLS để playback trên web
